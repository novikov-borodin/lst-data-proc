# lst-data-proc
LS-system data processing with the LST-method

Input data: fS.dat - superposition data (S), fW.dat - distorted data (W=S*V), fV.dat - data to be reconstructed (V)

fS - rectangular pulse, fSa - accidental pulse, fSt - triangular pulse, fSe - triangular-like pulse with exponential edges

Main program (Maple 2017): LST-main.mw uses file with subroutines: LST-sub.m (created by LST-sub.mw)

Load files in separate directory and run LST-main.mw file in Maple 2017 or higher.
